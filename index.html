Welcome to Gomoku!

Game rules:

  • 2 Players - black vs white
  • Each player takes it in turn to place a stone on the board
  • First player to reach 5+ unbroken stones in a row horizontally, vertically, or diagonally wins

You will need the other players game ID in order to join a game

This Gomoku game is totally free to play, its code is open source and all game data is open.

The backend board game engine is on GitHub BoardGameEngine

The frontend HTML5/Canvas Gomoku board is also on GitHub BoardGameEngine
